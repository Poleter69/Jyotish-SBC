# Jyotish SBC

Sarvatobhadra Chakra — Vedic Transit Analysis App

A complete 9×9 SBC grid with Vedha analysis, built as a mobile-optimized web app.

## Features
- Full 9×9 Sarvatobhadra Chakra grid
- All 28 Nakshatras including Abhijit
- 4 Ayanamsa — Pushya Paksha, Lahiri, B.V. Raman, Sri Yukteswar
- Planet Vedha analysis
- Works fully offline

## Live App
https://poleter69.github.io/Jyotish-SBC


## Tech
Pure HTML + CSS + JS. No frameworks. No dependencies.
